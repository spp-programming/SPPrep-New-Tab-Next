What's New

Last updated as of version 4.2. This page reflects all changes made up to the currently installed version, but not newer ones.
If you're looking for an update to a newer version, you can restart your device (at least on school chromebooks) if you don't want to wait around for an automatic update.
If you are seeing a new "footer" below SPP New Tab (with the "Customize Chrome" button), here's how to turn that off.

Version 4.2 (Latest, installed version)

This is a minor release because it does not contain any new major features.

🎉 New features:

  • Secret! 🎆 Added Missing Texture background.
  • Secret! 📼 Videos can now be used as a custom background.
  • 💁 A tour of the basic features is now available. You can find it in the Menu at the top left when you're on a new tab.
  • ☁️ Settings can now be stored in the cloud (using Chrome/Firefox sync).
  • 🅱️ Monograms can now be generated for custom links (if you don't want to upload an icon)
  • 💾 More fields are now collected in the support data.
  • 🗣️ More clear and descriptive error messages and wording.
  • ✅ Other general quality-of-life improvements.

🧑‍💻 Behind the scenes:

  • 🚀 The split layout is now implemented more cleanly and loads faster
  • 🛟 More error handling, especially when saving and loading
  • 🦊 Introduced preliminary support for Firefox
  • 🧹 Some code cleanup and added JSDoc annotations

✅ Fixed small bugs:

  • 🚪 The "Hide Club Hub button" and custom link name inputs in Settings now trigger the unload handler
  • Secret! ↩️ An error is now shown in the passcode modal when you get it wrong, like it did in version 3.x.
  • 🎞️ Custom button images are no longer cut off when using images with a non-1:1 aspect ratio.

Version 4.0 (Previous previous version)

This is a major release. That's a bit of an understatement.

Version 4.0 represents a full rewrite of SPP New Tab with a more modular, maintainable codebase. Here's what's new in this release:

🚨 Fixed major bugs:

  • 📡 Letter day lookup is better optimized and tolerant to errors

🎉 New features:

  • 🙈 The School Calendar button can now be hidden.
  • 💠 Custom links to your favorite sites can now be added.
  • 🕒 The clock can now be set to 24-hour time or 12-hour time with the AM/PM part visible.
  • 🆕 The split layout from the staff version is now available.
    • 👉 Used by default in the staff version, but not in the student version. The layout type can be switched on both versions of the extension.
  • Secret! ⚙️ Secret Settings is now a dedicated page.
  • Secret! 📤 You can now upload a custom background in Secret Settings.
  • Secret! 🔀 Rickroll background has been replaced with one that cycles through colors
  • 📦 The "What's New" and "Bell Schedule" buttons are now smaller and have been moved to the bottom corners.
    • 👉 There's also a new "Menu" button at the top left to access the Settings and About pages.
    • 👉 The Bell Schedule option now offers to enlarge the Bell Schedule (by opening it in another tab)
  • 🍺 Navigation bar for Settings, What's New, and About pages
  • 🖱️ You can now click on the extension icon in the toolbar
  • ✨ The logo on the top right corner has been updated with a higher-quality version
  • 🆕 Club Hub shortcut has been added. It can be hidden if you want.

🧑‍💻 Behind the scenes:

  • ✨ A more modular codebase using ECMAScript Modules
  • 🅱️ Bootstrap is now used for the entire interface.
  • 🗃️ Extension storage is now used for storing settings, instead of localStorage. Existing data stored in localStorage is migrated automatically.
  • 🤖 The staff and student versions of the extensions are now part of the same codebase, so staff/student specific functionality is handled using a config file.
  • 🕒 The clock is slightly more precise, responding to changes about 0.5 ms faster (by using textContent instead of innerHTML)

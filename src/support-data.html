Collect support data

This data may be requested by our team if we need it for debugging an issue report. You can file an issue report by clicking on "GitHub Issues" in the navbar.

The support data file contains all data stored by the extension, which includes all settings, custom links, and backgrounds. Keep in mind that anybody with this file will be able to know which background you've set and the custom links that you've added.

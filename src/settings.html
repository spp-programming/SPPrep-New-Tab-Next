Buttons

If you are subscribed to the School Calendar in your Google Calendar, the dedicated School Calendar button becomes redundant. So you can disable it here if you want.

Custom links allow you to add your own buttons to SPP New Tab, linking you to your favorite sites. You can add up to three.

Custom link #1

Required fields are marked with an asterisk (*). They are only required when this custom link is enabled.
Click on the button above to upload an icon. The icon you select will be previewed inside the button so you can see how it will look. The maximum file size is 512KB (512000 bytes).
You can also reset the icon to the default one.
Enter the name of this custom link. It will be displayed when you hover over the button.
Enter the URL of this custom link. This is where the button leads to when you click on it.
